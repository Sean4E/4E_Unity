What is Vuforia?

Vuforia is an image-based AR engine. Your camera recognises printed images and overlays 3D content on top of them. Unlike AR Foundation which detects surfaces, Vuforia locks content to specific images you choose.

Vuforia is like a QR code scanner for 3D. Print an image, point your camera at it, 3D content appears anchored to it. Move the image, the 3D content follows.

The killer feature: Vuforia works in the Unity Editor with your webcam. No phone build required. Press Play, hold up your printed image, and see AR content immediately.

Use cases: product demos, museum exhibits, educational materials, business cards, packaging, textbook augmentation.

Account Setup

Step 1: Go to developer.vuforia.com and create a free account.

Step 2: Navigate to License Manager → Get Development Key. Give it a name and copy the key.

License types: Basic (free) = unlimited for development and testing. Classic = paid for commercial distribution. Start with Basic — it's all you need.

Step 3: Go to Target Manager → Add Database → choose Device type.

Step 4: Click into your database → Add Target → upload an image file (JPG/PNG).

Vuforia will analyse your image and give it a star rating (0–5). Higher = better tracking.

Multiple Targets

Add multiple ImageTarget GameObjects — each linked to a different image in your database. Each target shows different 3D content.

Increase simultaneous tracking: In the Vuforia Configuration, set Max Simultaneous Tracked Images from 1 to 4.

Detect which target is active:

C# — MultiTargetHandler.cs 
using UnityEngine;
using Vuforia;

public class MultiTargetHandler : MonoBehaviour
{
    void Start()
    {
        var observers = FindObjectsOfType<ObserverBehaviour>();
        foreach (var obs in observers)
            obs.OnTargetStatusChanged += OnStatusChanged;
    }

    void OnStatusChanged(ObserverBehaviour behaviour, TargetStatus status)
    {
        string targetName = behaviour.TargetName;
        bool tracked = status.Status == Status.TRACKED ||
                       status.Status == Status.EXTENDED_TRACKED;

        Debug.Log($"{targetName}: {(tracked ? "Found" : "Lost")}");
    }
}

Animations on Target Found

Play an animation when the target is detected, and stop it when the target is lost. Use OnTargetStatusChanged to trigger animations.

C# — TargetAnimator.cs 
using UnityEngine;
using Vuforia;

public class TargetAnimator : MonoBehaviour
{
    public Animator animator;
    public string triggerName = "Play";

    void Start()
    {
        GetComponent<ObserverBehaviour>()
            .OnTargetStatusChanged += OnStatus;
    }

    void OnStatus(ObserverBehaviour b, TargetStatus status)
    {
        bool found = status.Status == Status.TRACKED;

        if (found)
        {
            animator.SetTrigger(triggerName);
            // Scale-up entrance effect
            StartCoroutine(ScaleIn());
        }
    }

    System.Collections.IEnumerator ScaleIn()
    {
        transform.localScale = Vector3.zero;
        float t = 0;
        while (t < 1f)
        {
            t += Time.deltaTime * 3f;
            transform.localScale = Vector3.Lerp(
                Vector3.zero, Vector3.one,
                Mathf.SmoothStep(0, 1, t));
            yield return null;
        }
    }
}

08

UI Overlay

Add a Canvas (Screen Space — Overlay) for instructions and status. Show "Scanning..." when no target is found, and "Target Found!" when tracked.

See the Canvas Journey for full UI details. Here we cover the Vuforia-specific bits.
C# — ARUIController.cs 
using UnityEngine;
using TMPro;
using Vuforia;

public class ARUIController : MonoBehaviour
{
    public TMP_Text statusText;
    public GameObject scanningPanel;

    void Start()
    {
        var observer = FindObjectOfType<ObserverBehaviour>();
        observer.OnTargetStatusChanged += OnStatus;
        scanningPanel.SetActive(true);
    }

    void OnStatus(ObserverBehaviour b, TargetStatus status)
    {
        bool tracked = status.Status == Status.TRACKED;
        statusText.text = tracked ? "Target Found!" : "Scanning...";
        statusText.color = tracked ? Color.green : Color.white;
        scanningPanel.SetActive(!tracked);
    }
}

Ground Plane (Bonus)

Vuforia also supports plane detection (similar to AR Foundation). This lets users tap the ground to place objects without needing a printed image.

  2. GameObject → Vuforia Engine → Ground Plane → Ground Plane Stage

C# — GroundPlacer.cs 
using UnityEngine;
using Vuforia;

public class GroundPlacer : MonoBehaviour
{
    public GameObject prefab;
    public PlaneFinderBehaviour planeFinder;
    GameObject placed;

    public void OnContentPlaced()
    {
        if (placed != null) Destroy(placed);
        placed = Instantiate(prefab,
            planeFinder.PlaneIndicator.transform.position,
            planeFinder.PlaneIndicator.transform.rotation);
    }
}
Ground Plane detection works best on textured surfaces — plain floors may not detect well. Carpet, wood grain, and tile patterns work best.

Build & Deploy

Vuforia apps can be built to Android and iOS just like AR Foundation. The key difference: Vuforia also works in the Editor, so test there first!

Android build:

1. File → Build Settings → Android
2. Player Settings → Min API Level: API 24
3. Scripting Backend: IL2CPP
4. Target Architectures: ARM64

5. Connect phone via USB
6. Enable Developer Mode + USB Debugging
7. Build And Run

iOS build: Same process as AR Foundation — builds an Xcode project, sign with Apple Developer account, deploy to device.
